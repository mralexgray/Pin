1 Pin
2 ===
3 An easy to use Arduino library for fast I/O using port manipulation. Supports Arduino Mega, Arduino Uno, and Arduino Leonardo. Complete documentation can be found [here](https://fenichelar.github.io/Pin/index.html). Common uses are listed below.
4 
5 ## Import Pin Library
6 ```C
7 #include <Pin.h>
8 ```
9 
10 ## Create Pin Object
11 Single Pin
12 ```C
13 Pin myPin = Pin(5);
14 ```
15 Array of Pins
16 ```C
17 Pin myPins[] = {6,7};
18 ```

20 ## Use as Input
21 Set Mode to Input
22 ```C
23 myPin.setInput();
24 ```
25 Enable/Disable Pullup Resistor
26 ```C

29 ```C

31 ```

34 Set Mode to Output

40 myPin.setHigh();

46 ## Get Pin Info
47 Get Mode (INPUT/OUTPUT)
48 ```C
49 myPin.getMode();
50 ```
51 Get State (HIGH/LOW)

53 myPin.getState();

63 myPin.toggleMode();
